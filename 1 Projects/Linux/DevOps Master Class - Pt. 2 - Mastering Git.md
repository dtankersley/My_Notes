** Why Version Control? **

* Even as a team of one we commonly have multiple versions of the same file as we work on it over time
		* A version control system provides capabilities to centrally store & track our content with metadata like versions, comments, etc
* When you expand to more than one, it becomes a necessity


Benefits & Features of Version Control systems

* the source of truth for the company/project 
* Enables collaberation
* "Encourages" consistency
* History and immutability
	* Hooks into additional security 
* CI/CD integration

Types of version Control systems

* Centralized
* distributed
*

What is Git?
  * a free, Open-source, distributed version control system
  * Utilizes folders and files on the Local file system
  *   Created by Linus Torvalds\the de-facto standard 
  *  Used by many repo solutions like Github, Azure Devops & Bitbucket
