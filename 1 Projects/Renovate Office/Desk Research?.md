# Standing Desk Research

## Requirements

- Minimum 160cm wide (dual monitor + laptop)
- Electric sit/stand with memory presets
- Stable enough that monitors don't wobble at standing height

## Options Considered

| Desk | Width | Price | Notes |
| ---- | ----- | ----- | ----- |
| IKEA BEKANT | 160cm | €480 | Affordable but no memory presets |
| Flexispot E7 | 180cm | €520 | Great reviews, 4 memory presets |
| Fully Jarvis | 180cm | €680 | Premium but ships from US |

## Decision

Going with the **Flexispot E7** — best balance of price, features, and availability in the EU. Ordered the bamboo top (180x80cm).

## Standing Desk Tips

- Anti-fatigue mat is a must
- Start with 30 min standing, 60 min sitting — build up gradually
- Keep the monitor at eye level in both positions
